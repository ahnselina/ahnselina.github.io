整数集合相关API

函数 作用 时间复杂度
intsetNew 创建一个新的整数集合 O(1)
intsetAdd 将指定元素添加到整数集合中 O(N)
intsetRemove 从整数集合中移除指定元素 O(N)
intsetFind 查找指定的值是否在集合中 因数组有序，可通过二分查找法来进行，故复杂度为O(logN)
intsetRandom 从整数集合中随机返回一个元素 O(1)
intsetGet 获取指定索引上的元素 O(1)
intsetLen 返回整数集合包含的元素个数 O(1)
intsetBlobLen 返回整数集合占用的内存字节数 O(1)

整数集合特点

  • 新建整数集合时，其encoding是int16_t类型
  • 只支持升级，不支持降级
  • 查找使用二分查找法
  • 整数集合的底层实现为数组，这个数组以有序、无重复的方式保存集合元素， 在有需要时， 程序会根据新添加元素的类型，改变这个数组的类型。
  • 升级操作为整数集合带来了操作上的灵活性， 并且尽可能地节约了内存。

小端模式与大端模式
小端模式：按照从低地址到高地址的顺序，依次存放数据的低字节到高字节
大端模式：按照从低地址到高地址的顺序，依次存放数据的高字节到低字节
详细介绍可参考计算机中的小端模式和大端模式

大端模式 0x12 0x34 0xab 0xcd
